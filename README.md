# kakao-techcamp

이 프로젝트는 2023년 kakao-techcampus의 학습과 프로잭트를 저장한 저장소 입니다.

See [Demo](https://www.google.com/)

## Documentation

### Installation

```shell
$ git clone {repo url}
$ cd {reponame}
$ pip install -r requirments.txt
```

### How To Start

```shell
$ python main.py
```

### Dependency

- Python==3.10
- django==3.0.0

### Features

- 행운의 숫자 추출
- 행운의 숫자 저장
- 행운의 숫자 기반 구매
